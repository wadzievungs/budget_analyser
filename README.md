# Budget Analyser 

A Python data analysis tool that reads financial transaction data from a CSV file and produces a clear budget summary. Built as Project 2 of my FinTech portfolio.

## What it does 
- Reads transaction data from a CSV file 
- Calculates total income, total expenses and net savings
- Breaks down spending by category sorted from highest to lowest
- Identifies the highest spending category

## Technologies used
- Python 3
- pandas

## How to run it
1. Clone this repository
2. Install dependencies: pip install pandas
3. Navigate to the src folder
4. Run: python main.py
